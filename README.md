# 🤖 Universal Document RAG Assistant

An AI-powered system that allows users to upload documents (PDF/TXT) and ask questions about them using natural language.

---

## 🚀 Features
- 📄 Upload PDF/TXT documents
- 🔍 Semantic search using embeddings
- 💬 Chat interface (Gradio)
- ⚡ FastAPI backend
- 📚 Source-based answers (with references)

---

## 🧠 Tech Stack
- Python
- FastAPI
- Gradio
- LangChain
- ChromaDB / FAISS
- HuggingFace Transformers
